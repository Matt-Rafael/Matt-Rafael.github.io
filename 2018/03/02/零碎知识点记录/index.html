零碎知识点记录

发表于 2018-03-02 | 分类于 Python学习

该篇文章主要是包含Python的知识点

name变量

  • 当文件是被调用时，name的值为模块名；
  • 当文件被执行时，name的值为 ‘__main__’

特殊的格式

  • r''表示''内部的字符串默认不转义
  • '''xxx'''格式可以表示多行内容

整数的除法

  • /除法计算结果是浮点数
  • //地板除法结果是整数

字符串处理

  • ord() 获取字符的整数表示，chr()把编码转换为对应的字符

  • 字符串和字节(b前缀)转换：

    b'abc'.decode('utf-8')，从网络或磁盘上读取的是字节流
'abc'.encode('utf-8')

  • 字符串的格式化

    'Hello, %s' % 'World'
'Hello, {0}'.format('World')

复杂的数据类型

  • 列表：['a', 'b']
  • 元组：　('a','b')
  • 字典：['a': 'A', 'b': 'B']
  • 集合(set)：{'a', 'b'}

if 的简写

if var：如果var为零数值、空字符串、空列表、None，则为Flase

函数的参数

  • 默认参数

    def power(x, n = 1):
    return x * n

  • 可变参数

    def calc(*numbers):
    return len(number)

  • 关键字参数：允许传入０个或多个包含参数名的参数

    def person(name, age, **kw):
    print('keyword')

面向对象编程

  • 类的定义与使用：

    class Student(object):
    def __init__(self, name, score):
        self.name = name
            self.score = score
    def print_score(self):
        print('%s - %s' % (self.name, self.score))

student = Student('matt', 100)
student.print_score()

如果想要内部属性不被外部访问，可以把属性的名字前加上两个下划线__
